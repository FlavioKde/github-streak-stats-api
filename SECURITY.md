# Security Policy

## Supported Versions

Only the latest stable version of this project is actively supported.

| Version | Supported |
|--------|-----------|
| latest | ✅ |
| < latest | ❌ |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT open a public issue.

Instead, report it privately:

- Open a GitHub Security Advisory (preferred)
  https://github.com/FlavioKde/github-streak-stats-api/security/advisories/new

OR

- Create a private issue if advisories are not available

---

## What to include

When reporting a vulnerability, please include:

- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

---

## Response timeline

We aim to respond within a reasonable timeframe and will prioritize security issues over feature requests or bugs.

---

## Scope

This policy applies to:

- API endpoints
- GitHub integration logic
- caching system
- deployment configuration (Vercel)
